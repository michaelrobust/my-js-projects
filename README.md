## üöÄ How to Run

Each project can be run independently. For projects with dependencies:

```bash
# Navigate to specific project
cd [project-name]

# Install dependencies (if package.json exists)
npm install

# Start the application
npm start
# or
node app.js
```

For static websites:
```bash
# Simply open the HTML file in a browser
open index.html
```
